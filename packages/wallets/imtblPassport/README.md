# @layerswap/wallet-imtbl-passport

Immutable Passport wallet provider for the Layerswap Widget. Enables Immutable Passport wallet connections for Immutable X ecosystem.

## Installation

```bash
npm install @layerswap/wallet-imtbl-passport @layerswap/wallet-evm wagmi viem @tanstack/react-query
# or
yarn add @layerswap/wallet-imtbl-passport @layerswap/wallet-evm wagmi viem @tanstack/react-query
# or
pnpm add @layerswap/wallet-imtbl-passport @layerswap/wallet-evm wagmi viem @tanstack/react-query
```

## Documentation

For setup instructions, configuration details, and usage examples, see the [official Immutable Passport documentation](https://docs.layerswap.io/integration/UI/Widget/WalletManagement/ImmutablePassportProvider).

**Note:** Immutable Passport requires the EVM provider as a dependency and must be used alongside it.

## Features

- Immutable Passport wallet connection
- Integration with Immutable SDK
- Support for Immutable X transactions
- Seamless onboarding for gaming-focused wallet solution
- Requires EVM provider as a dependency

## TypeScript

This package includes TypeScript definitions. All types are exported from the main entry point.
